Div elements are block by default (unless changed)
Spans are inline element
Block elements ex:(div,section,nav,article,h1,paragraphs,etc) can be used to create a block of text or to arrange boxes into a layout
Block elements stack ontop of each other while inline elements line up next to each other

A block element will expand to fill the width of it's container
